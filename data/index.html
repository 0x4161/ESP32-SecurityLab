# What is REAPER?

A pocket-sized wireless attack platform that runs entirely on an ESP32-WROOM-32, controlled by a single button and a 0.96″ OLED. No phone, no laptop, no app — every operation is driven from the device itself.

Built around raw 802.11 frame injection and bare ESP-IDF radio control, it harvests Wi-Fi handshakes, kicks clients off APs, floods the spectrum with phantom beacons, and brute-forces PSKs from a built-in wordlist.

# Arsenal

📡

Deauth / Disassoc

Raw 802.11 frame injection on the target’s channel. Single or multi-AP rotation.

🌐

Beacon Flood

Sprays the spectrum with thousands of phantom SSIDs. Sequentially numbered.

🔑

PSK Brute-Force

5000-password built-in wordlist. Iterates against any visible WPA/WPA2 SSID.

👻

Evil Twin

Clones a target SSID as an open AP on the same channel.

📶

Probe Sniffer

Captures probe requests revealing every network nearby devices remember.

🤝

EAPOL Sniffer

Captures the WPA 4-way handshake for offline cracking with hashcat.

🔵

BLE Spam

Apple proximity / Microsoft Swift Pair / Samsung Easy Setup payloads.

🛰

Wi-Fi / BLE Scan

Full 2.4 GHz enumeration plus BLE advertising discovery.

# Control surface

A single BOOT button (GPIO0) drives the entire menu. All operations are visible on the OLED in real time.
